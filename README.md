# WhatsApp Bot V6 - Termux Compatible

هذا البوت مصمم للعمل على تطبيق **Termux** ويدعم استدعاء رمز QR بالطريقة الحديثة.

## 🚀 طريقة التثبيت على Termux

افتح تطبيق Termux وانسخ الأوامر التالية بالترتيب:

### 1. تحديث الحزم وتثبيت المتطلبات الأساسية
```bash
pkg update && pkg upgrade -y
pkg install nodejs git -y
```

### 2. تحميل البوت من GitHub
```bash
git clone <YOUR_REPO_URL>
cd whatsapp-bot
```

### 3. تثبيت المكتبات المطلوبة
```bash
npm install
```

### 4. تشغيل البوت
```bash
node index.js
```

---

## 📸 طريقة ربط الحساب (QR Code)
عند تشغيل البوت لأول مرة، سيظهر رمز **QR** في الشاشة.
1. افتح واتساب على هاتفك.
2. اذهب إلى **الأجهزة المرتبطة (Linked Devices)**.
3. اختر **ربط جهاز (Link a Device)**.
4. امسح الرمز الظاهر في Termux.
